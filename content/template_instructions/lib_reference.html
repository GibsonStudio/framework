lib.js Reference

lib.js is automatically included with every file.

root_path ()

returns the path to the root of the project site. The default value is '../../'.

script_path ()

returns the path to the javascript folder.

image_path ()

returns the path to the images folder.

css_path ()

returns the path to the css folder.

write_head ()

Writes the HTML <head>. Produces:

write_header ()

Writes the HTML for the page header, including, logo, title, page number and navigation buttons. Should be included on every page.

write_menu_header ()

Writes the HTML for the page header, including, logo only.

include_css (filename)

Includes a css file located in the root/css folder.

Example Usage:

include_javascript (filename)

Includes a javascript file located in the root/javascript folder.

Example Usage:

write_nav ()

Adds the Quick Navigation menu. Should be included on every page.

write_debug ()

Adds the output div for the debug function to write to.

debug (string)

Outputs string to the debug window.

Note: The debug output div must have been added with the function write_debug().

add_audio (name, args)

Adds an audio element to the page. See sync_to_audio.html for more details.

add_video (name, args)

Adds a video element to the page. See video.html for more details.

media_finished ()

This function is called when a video or audio object has finished playback if object.onended was set to true. The function highlights the Next Scene button.

pause_media ()

Pauses / resumes any video with id="my_video" or audio with id="my_audio". Can be manually called, or is called by the keyboard shortcut 'P'.

get_url_param (param, def)
  • param: String. Name of parameter.
  • def: String. Default value of parameter.

Gets the value of a parameter in the url, i.e. index.html?param=value.

returns: string

button_nav ()

Called when the Quick Navigation button is clicked.

button_back ()

Called when the Back button is clicked.

button_next ()

Called when the Next button is clicked.

keyboard_shortcuts (e)

Called when any key is pressed. The event listener is assigned by scene_loaded():

scene_loaded ()

Called by body onload:

Does the following:

  • Checks the current scene and dims Next / Back buttons accordingly.
  • Adds an event listener for keyboard shortcuts.
  • Sets up audio events if the scene is using 'sync to audio'.
get_scene_number (current_scene, scenes)
  • current_scene: String. Filename of current loaded scene, e.g. index.html.
  • scenes: Array. The 'my_scenes' array.

This returns the current scene index, i.e. the index of the scene in the my_scenes array.

returns: integer

is_in_array (value, array)
  • value: Mixed. Value to look for in an array.
  • array: Array. The array to search in.

returns: boolean

Note: The following are used when syncing HTML to audio. This is explained in detail here: sync_to_audio.html

set_up_audio_events ()

Initializes 'sync to audio'.

get_object_max ()

Examines the 'frames' array and returns the highest object id number. There is a for loop that runs and processes the visibility of objects - we need to know how many times to run this loop.

returns: integer

show_frame (f)
  • f: Integer. Frame to show - index in frames array.
show_element (my_element)
  • my_element: String. ID of an element.
hide_element (my_element)
  • my_element: String. ID of an element.
get_frame_from_time (t)
  • t: Integer. Time in seconds.

Examines the array frames and works out which frame to show based on the value of t.

returns: integer
