Lesson Maker

I have created a 'Lesson Maker' which can generate ini_lesson.js and the required html files to display video. 'Lesson Maker' reads from an existing lesson .xml file, e.g. AC-int-pow.xml.

'Lesson Maker' is only intended for lessons that previously existed as Flash and have been converted to video.

Installing Lesson Maker

Lesson Maker is built using PHP (javascript does not have permission to create files). For this reason you will need to run Lesson Maker from a local webserver (e.g. WAMP or XAMPP).

  1. Copy the files from L:\cbt_video_source\lesson_maker to a folder on your local web server, e.g. C:\wamp\www\lesson_maker
  2. Make sure you have a folder output in your local web site folder.
  3. Go to http://localhost/lesson_maker/ to run.
Using Lesson Maker

For this example we are going to make files for the lesson AC-alt-gen:

  1. Open the file L:\cbt_published\atpl_ac electrics\content\AC-alt-gen.xml in Notepadd++ (or your text editor of choice).
  2. Copy the contents of AC-alt-gen.xml and paste into the Input (XML) filed of Lesson Maker.
  3. Click Create lesson Files.
  4. Check the Output field for any errors.

You'll now find a new folder AC-alt-gen in C:\wamp\www\lesson_maker\output\. This folder will contain the files:

  • ini_lesson.js
  • index.html
  • AC-alt-gen-s2.html
  • AC-alt-gen-s3.html
  • AC-alt-gen-s4.html
  • AC-alt-gen-s5.html
  • AC-alt-gen-s6.html
  • AC-alt-gen-s7.html
  • AC-alt-gen-s8.html
  • AC-alt-gen-s9.html
  • AC-alt-gen-s10.html
  • AC-alt-gen-s11.html

Note: AC-alt-gen-s1.html does not exist - it is called index.html as it is the default file for this lesson.

These files, along with the videos you created from Flash, are all that is needed to produce a new HTML5 video lesson.

Where to put these Files?

Copy (or Move) the folder AC-alt-gen from your output folder to L:\cbt_video_published\content. Add your video files to L:\cbt_video_published\content. That's it; lesson created.
