Simple Slider

Use the slider to control the value:

To use a slider:

  • Include slide.css: include_css('slider.css');
  • Add an input give it the following parameters:
    • type="range"
    • id="" - a unique id
    • value="" - the starting value
    • min="" - minimum value
    • max="" - maximum value
    • step="" - the slider value increments
  • Add the following event handlers: (these will cover all browsers)
    • oninput="slider_changed()"
    • onchange="slider_changed()"
  • Add some javascript to define the function slider_changed () {}. This will trigger if the slider value is changed.
  • call slider_changed() to initialize the slider
