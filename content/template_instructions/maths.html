Equations / Maths CSS

Equations can be displayed by surrounding your characters with tags and including math.css:

Tags

<math></math>: Container for an equation.

<mo></mo>: Operator, = or ≠

<mi></mi>: Symbol, x, y etc...

Fractions

Fractions are defined with the <mfrac> tag and must have a numerator <mn> and a denominator <md>.

Example: The following code produces:

x = 2 3
Powers

Powers are added using the <pow> tag:

a2 = b3 + c4
Powers 2 - 2 Line Powers

If a 2 line thing is raised to a power (e.g. a fraction) you need to use the <pow2> tag to account for the greater line height.

Example:

x = a b 2
Subscripts

The normal <sub> tags will break the maths css styles. Use <ss> instead:

a1 = b2 + c3
Brackets

Brackets are added using 2 different tags:

<bo></bo>: Open bracket.

<bc></bc>: Close bracket.

x = 1 2 ÷ 3
Square Root

Square roots are added with the <sqrt> tags.

Example:

2 ≈ 1.414
Small Inline Fractions

Small inline fractions e.g. 3/5 or 7/8 can be added using the tags:

<fn></fn>: Small fraction numerator.

<fd></fd>: Small fraction numerator.

Example:

7/8 = x 4

Note: You have to enter the dividing line as a / character.

Plain Text

You can use plain text in an equation using the tag <plain>:

Kinetic Energy = 1/2 m v 2
