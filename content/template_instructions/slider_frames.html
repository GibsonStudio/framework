Slider Controlling an Animation

Use the slider to control the value:

To use a slider to control an animation:

  • Include slide.css: include_css('slider.css');
  • Add an input give it the following parameters:
    • type="range"
    • id="" - a unique id
    • value="" - the starting value
    • min="" - minimum value
    • max="" - maximum value
    • step="" - the slider value increments
  • Add the javascript var frame_rate = 30;
  • Define function run_each_frame() {} - this will run every 1 second / frame_rate.
  • At the end on function run_each_frame() add setTimeout(run_each_frame, (1000.0 / frame_rate) ); - this calls the function again after an appropriate delay.
  • call run_each_frame() to initialize the animation
