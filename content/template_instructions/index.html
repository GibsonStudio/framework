Including 3rd Party Libraries

The following 3rd party libraries can be added with the following functions:

  • jQuery UI: include_jquery_ui(); - required making divs draggable.
  • jQuery Rotate: include_jquery_rotate(); - For rotating divs with javascript.
  • Raphael: include_raphael(); - For animateing along paths.
  • Three.js: include_threejs(); - For displaying 3D scenes (includes OrbitControls.js by default).

Note: jQuery is included by default.
