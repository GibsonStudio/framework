Video
We only need .mp4

We can display video using the HTML5 standard. To cover most browsers and devices we only need one video format, .mp4.

Fullscreen Video

To show a 'fullscreen' video, i.e. a normal video scene, you should add your video with the javascript:

The add_video function can take the following optional arguments - default values in brackets:

  • id: (my_video) - string. id for video element.
  • width: (false) - integer. Width of video.
  • height: (false) - integer. Height of video.
  • style: ('') - string. Any styles to add to the video element.
  • classes: ('') - string. Any additonal classes to add to the video element.
  • controls: (true) - boolean. Sets whether video has controls or not.
  • loop: (false) - boolean. Sets whether video loops or not.
  • autoplay: (true) - boolean. Sets whether video autoplays or not.
  • fullscreen: (true) - boolean. Sets whether video is 'fullscreen' or not.
  • onended: (true) - boolean. Whether or not to call the function media_finished() at end.
  • poster: (false) - string. File to use as poster image.
