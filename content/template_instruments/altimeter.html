The altimeter model is initialized by the variable:

var altimeter = {};
altimeter.altitude = altitude;
altimeter.pressure = pressure;
altitmeter.subscale = subscale;

The altimeter variable should have 3 properties:

  • .altitude: (float); This is the actual physical altitude of the instrument, NOT the altitude to be displayed.
  • .pressure: (float); The atmospheric air pressure.
  • .subscale: (float) 950 - 1050; The setting on the subscale.

The above code will initialize the altimeter model but it does not update the graphics on sceen. Graphics can be updated with the function:

draw_altimeter();

The function draw_altimeter(); will use the above 3 values and calculate the indicated alitude - this is the value displayed on the instrument.

Note: If you just want a simple model to display an altitude, set altitude.pressure and altitude.subscale to the same value (1000). You can then set altimeter.altitude to any required value and the function draw_altimeter() will display that altitude.
