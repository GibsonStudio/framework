E2B Compass
Direction:

The e2b model is initialized by the variable:

var e2b = {};
e2b.direction = direction;
e2b.min = 32;
e2b.max = -625;
  • .direction: (float); This is the direction (heading) to be displayed on the instrument.
  • .min: (int); The left position (left: Xpx;) to align the 1st North on the dial with the index line.
  • .max: (int); The left position (left: Xpx;) to align the 2nd North on the dial with the index line.

The above code will initialize the E2B model but it does not update the graphics on sceen. Graphics can be updated with the function:

draw_e2b();
